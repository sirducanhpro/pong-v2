About this Sketch

This is a Pong game
It is inspired by Allan Alcorn's 1972 Pong game. Read more about the game here.
I intend to implement more feaures to the game that would enhance it. Such features may include an A.I, more unique ball designs, different graphics..

Directions

  • The objective is to score as many points as possible by not letting the ball touch your edge
  • Players must move the paddle that reflect the ball
  • You can change the directions of the paddle up and dow using either W,S or UP and DOWN arrow key.
